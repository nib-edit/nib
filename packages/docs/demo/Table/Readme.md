### Keyboard shortcuts

| S.No. | Keyboard Shortcut | Description         |
| ----- | ----------------- | ------------------- |
| 1     | Cmd-T             | Insert table        |
| 2     | Tab               | Go to next cell     |
| 3     | Shift-Tab         | Go to previous cell |
